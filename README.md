# 2026 Asia Trip — Planning Hub

Group trip for **8–10 people**, **~2 weeks**, late November → early December 2026. Explorer-historian style (history + nature + unusual transport + viewpoints), independent and off-peak, within a **10,000 PLN per person** flights-plus-accommodation budget. Whole group flies together from **Warsaw on one Turkish Airlines open-jaw ticket** (HAN-in / SGN-out, via Istanbul — Gulf-hub-free routing).

There are **two plans on the table.** Pick one — they share the same flights, hub, budget and travel style, and differ only in the middle of the route.

## How to read it
The trade is essentially: **Option A** keeps Vietnam's densest history leg (imperial citadels, Cham ruins, the great caves and the sleeper train). **Option B** swaps central Vietnam for South Cambodia — colonial decay, an off-track island and a beach finish — at the cost of those Vietnamese sights and the night train. Same start, same finish, same ticket either way.

## What's inside each folder
Both folders have the identical structure:

- `README.md` — decisions, flight routing, assumptions and caveats for that option.
- `*-itinerary.md` — the day-by-day route, transport table and logistics (main deliverable).
- `*-budget.xlsx` — editable per-person cost breakdown vs the 10,000 PLN budget.
- `daily-guide/index.html` — illustrated, bilingual (EN/PL) day-by-day visual guide.
- `route-map/index.html` — interactive route map; tap a stop for its transport leg and booking links.

Option B's README also lists **two blocking items to confirm before booking** (Vietnam multiple-entry e-visa for all travellers; Cambodia e-visa validity at the Prek Chak / Ha Tien land border).
